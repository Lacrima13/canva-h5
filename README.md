# 王紫玫作品集｜空白画布急诊室

## 部署方式

将本文件夹内所有文件上传到 GitHub 仓库根目录：

- `index.html`：作品集首页
- `work.html`：主作品页面
- `404.html`：路径错误回退页面
- `README.md`：说明文件

GitHub Pages 设置：

`Settings → Pages → Deploy from a branch → main / root`

## 提交链接

作品集链接：

`https://你的用户名.github.io/仓库名/`

作品链接：

`https://你的用户名.github.io/仓库名/work.html`

## 作品说明

《空白画布急诊室》是一款基于 Canva 可画品牌语境的 AIGC H5 交互作品。作品将设计痛点转化为“设计急诊”的剧情式交互体验，支持症状诊断、任务选择、实时编辑、配色切换、版式切换、Before/After 对比、作品卡导出和 Canva 可画跳转。
